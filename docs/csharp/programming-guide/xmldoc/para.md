# <a name="para-c-programming-guide"></a>\<para>(Guida per programmatori C#)

## <a name="syntax"></a>Sintassi

```xml
<para>content</para>
```

## <a name="parameters"></a>Parametri

- `content`

  Testo del paragrafo.
