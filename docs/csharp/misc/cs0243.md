# <a name="compiler-error-cs0243"></a><span data-ttu-id="2bb76-102">Errore del compilatore CS0243</span><span class="sxs-lookup"><span data-stu-id="2bb76-102">Compiler error CS0243</span></span>

<span data-ttu-id="2bb76-103">L'attributo Conditional non è valido per 'method' perché è un metodo di override</span><span class="sxs-lookup"><span data-stu-id="2bb76-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="2bb76-104">L'attributo <xref:System.Diagnostics.ConditionalAttribute> non è consentito in un metodo contrassegnato con la parola chiave [override.](../language-reference/keywords/override.md)</span><span class="sxs-lookup"><span data-stu-id="2bb76-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="2bb76-105">Per ulteriori informazioni, vedere [Sapere quando utilizzare le parole chiave Override e New](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="2bb76-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="2bb76-106">Il compilatore non esegue mai l'associazione ai metodi di override.</span><span class="sxs-lookup"><span data-stu-id="2bb76-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="2bb76-107">Viene associato solo al metodo di base e Common Language Runtime chiama l'override, in base alle esigenze.</span><span class="sxs-lookup"><span data-stu-id="2bb76-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="2bb76-108">Il codice seguente genera l'errore CS0243:</span><span class="sxs-lookup"><span data-stu-id="2bb76-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
