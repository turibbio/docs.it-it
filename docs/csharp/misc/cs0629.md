# <a name="compiler-error-cs0629"></a><span data-ttu-id="c7cc0-102">Errore del compilatore CS0629</span><span class="sxs-lookup"><span data-stu-id="c7cc0-102">Compiler error CS0629</span></span>

<span data-ttu-id="c7cc0-103">Il membro condizionale 'member' non può implementare il membro di interfaccia 'base class member' nel tipo 'Type Name'</span><span class="sxs-lookup"><span data-stu-id="c7cc0-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
<span data-ttu-id="c7cc0-104">[L'attributo Conditional](../language-reference/attributes/general.md#conditional-attribute) non può essere utilizzato nell'implementazione di un'interfaccia.</span><span class="sxs-lookup"><span data-stu-id="c7cc0-104">The [Conditional attribute](../language-reference/attributes/general.md#conditional-attribute) cannot be used on the implementation of an interface.</span></span>  
  
<span data-ttu-id="c7cc0-105">L'esempio seguente genera l'errore CS0629:</span><span class="sxs-lookup"><span data-stu-id="c7cc0-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
