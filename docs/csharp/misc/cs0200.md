## <a name="examples"></a><span data-ttu-id="83ab6-107">Esempi</span><span class="sxs-lookup"><span data-stu-id="83ab6-107">Examples</span></span>
<span data-ttu-id="83ab6-108">L'esempio seguente genera l'errore CS0200:</span><span class="sxs-lookup"><span data-stu-id="83ab6-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="83ab6-109">Nell'esempio seguente [vengono utilizzate proprietà implementate automaticamente](../programming-guide/classes-and-structs/auto-implemented-properties.md), [inizializzatori](../programming-guide/classes-and-structs/object-and-collection-initializers.md)di oggetto e viene comunque generato l'errore CS0200:</span><span class="sxs-lookup"><span data-stu-id="83ab6-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="83ab6-110">L'assegnazione a una proprietà o a un indicizzatore 'proprietà' di sola lettura può essere ottenuta tramite l'aggiunta di una funzione di accesso set o l'assegnazione alla proprietà nel costruttore dell'oggetto.</span><span class="sxs-lookup"><span data-stu-id="83ab6-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp

{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
