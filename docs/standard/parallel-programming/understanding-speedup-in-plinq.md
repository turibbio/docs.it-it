<span data-ttu-id="cf94a-103">L'obiettivo principale di PLINQ consiste nel velocizzare l'esecuzione di query LINQ to Objects mediante l'esecuzione di delegati di query in parallelo in computer multicore.</span><span class="sxs-lookup"><span data-stu-id="cf94a-103">The primary purpose of PLINQ is to speed up the execution of LINQ to Objects queries by executing the query delegates in parallel on multi-core computers.</span></span> <span data-ttu-id="cf94a-104">PLINQ assicura le prestazioni migliori quando l'elaborazione di ogni elemento in una raccolta di origine è indipendente, senza stati condivisi tra i singoli delegati.</span><span class="sxs-lookup"><span data-stu-id="cf94a-104">PLINQ performs best when the processing of each element in a source collection is independent, with no shared state involved among the individual delegates.</span></span> <span data-ttu-id="cf94a-105">Tali operazioni sono comuni in LINQ to Objects e PLINQ e spesso vengono definite "*squisitamente parallele*" perché si prestano facilmente alla pianificazione su più thread.</span><span class="sxs-lookup"><span data-stu-id="cf94a-105">Such operations are common in LINQ to Objects and PLINQ, and are often called "*delightfully parallel*" because they lend themselves easily to scheduling on multiple threads.</span></span> <span data-ttu-id="cf94a-106">Tuttavia, non tutte le query sono costituite interamente da operazioni squisitamente parallele. <span data-ttu-id="cf94a-120">Il risultato è un aumento della velocità di 750 millisecondi.</span><span class="sxs-lookup"><span data-stu-id="cf94a-120">This yields a speedup of 750
