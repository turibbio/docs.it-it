## <a name="disable-the-strong-name-bypass-feature-for-all-applications"></a><span data-ttu-id="63269-120">Disabilitare la funzionalità di bypass con nome sicuro per tutte le applicazioni</span><span class="sxs-lookup"><span data-stu-id="63269-120">Disable the strong-name bypass feature for all applications</span></span>  
  
- <span data-ttu-id="63269-121">Nei computer a 32 bit creare nel Registro di sistema una voce DWORD con valore 0 denominata `AllowStrongNameBypass` nella chiave HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\\.NETFramework.</span><span class="sxs-lookup"><span data-stu-id="63269-121">On 32-bit computers, in the system registry, create a DWORD entry with a value of 0 named `AllowStrongNameBypass` under the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\\.NETFramework key.</span></span>  
  
- <span data-ttu-id="63269-122">Nei computer a 64 bit creare nel Registro di sistema una voce DWORD con valore 0 denominata `AllowStrongNameBypass` nelle chiavi HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\\.NETFramework e HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\\.NETFramework.</span><span class="sxs-lookup"><span data-stu-id="63269-122">On 64-bit computers, in the system registry, create a DWORD entry with a value of 0 named `AllowStrongNameBypass` under the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\\.NETFramework and HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\\.NETFramework keys.</span></span>  
  
## <a name="disable-the-strong-name-bypass-feature-for-a-single-application"></a><span data-ttu-id="63269-123">Disabilitare la funzionalità di bypass con nome sicuro per una singola applicazione</span><span class="sxs-lookup"><span data-stu-id="63269-123">Disable the strong-name bypass feature for a single application</span></span>  
  
1. <span data-ttu-id="63269-124">Aprire o creare il file di configurazione dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="63269-124">Open or create the application configuration file.</span></span>  
  
    <span data-ttu-id="63269-125">Per altre informazioni su questo file, vedere la sezione file di configurazione dell'applicazione in [configurare le app](../../framework/configure-apps/index.md).</span><span class="sxs-lookup"><span data-stu-id="63269-125">For more information about this file, see the Application Configuration Files section in [Configure apps](../../framework/configure-apps/index.md).</span></span>

    ```xml  
    <configuration>  
      <runtime>  
        <bypassTrustedAppStrongNames enabled="false" />  
      </runtime>  
    </configuration>  
    ```  
  
 <span data-ttu-id="63269-127">È possibile ripristinare la funzionalità bypass per l'applicazione rimuovendo l'impostazione del file di configurazione o impostando l'attributo su `true` .</span><span class="sxs-lookup"><span data-stu-id="63269-127">You can restore the bypass feature for the application by removing the configuration file setting or by setting the attribute to `true`.</span></span>
