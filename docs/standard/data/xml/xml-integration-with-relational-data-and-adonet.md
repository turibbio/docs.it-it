- <span data-ttu-id="6faf9-109">È possibile associare la parte strutturata di un documento XML a un dataset per aumentare l'efficacia delle operazioni di archiviazione, indicizzazione e ricerca.</span><span class="sxs-lookup"><span data-stu-id="6faf9-109">The structured portion of an XML document can be mapped to a dataset, and be efficiently stored, indexed, and searched.</span></span>
