### <a name="the-connectionstrings-section"></a><span data-ttu-id="ef299-112">Sezione connectionStrings</span><span class="sxs-lookup"><span data-stu-id="ef299-112">The connectionStrings Section</span></span>

```xml  
  <configuration>  
    <connectionStrings>  
      <clear />  
      <add name="Name"
       providerName="System.Data.ProviderName"
       connectionString="Valid Connection String;" />  
    </connectionStrings>  
  </configuration>  
```
