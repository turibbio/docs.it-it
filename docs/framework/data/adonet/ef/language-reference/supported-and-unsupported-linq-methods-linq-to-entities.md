# <a name="supported-and-unsupported-linq-methods-linq-to-entities"></a>Metodi LINQ supportati e non supportati (LINQ to Entities)
In questa sezione vengono fornite informazioni sugli operatori di query standard LINQ (Language Integrated Query) supportati o non supportati nelle query LINQ to Entities. Numerosi operatori di query standard LINQ dispongono di una versione di overload che accetta un argomento integer. L'argomento integer corrisponde a un indice in base zero nella sequenza utilizzata, <xref:System.Collections.Generic.IEqualityComparer%601> o <xref:System.Collections.Generic.IComparer%601> . Se non specificato diversamente, queste versioni di overload degli operatori di query standard LINQ non sono supportate e quando si tenta di usarle viene generata un'eccezione.  
  
## <a name="projection-and-restriction-methods"></a>Metodi di proiezione e di restrizione  
 La maggior parte dei metodi di proiezione e restrizione LINQ è supportata nelle query di LINQ to Entities, ad eccezione di quelle che accettano un argomento posizionale. Per altre informazioni, vedere [operatori di query standard in query LINQ to Entities](standard-query-operators-in-linq-to-entities-queries.md). ## <a name="type-methods"></a>Metodi di tipo  
 Gli operatori di query standard LINQ che gestiscono la conversione dei tipi CLR e i test sono supportati nell'Entity Framework. Solo i tipi CLR mappati ai tipi del modello concettuale sono supportati in LINQ to Entities. Per un elenco dei tipi di modello concettuale, vedere [tipi di modelli concettuali (CSDL)](/ef/ef6/modeling/designer/advanced/edmx/csdl-spec#conceptual-model-types-csdl).
