# <a name="ememoryavailable-enumeration"></a>Enumerazione EMemoryAvailable
Contiene valori che indicano la quantità di memoria fisica disponibile nel computer. Questi valori vengono mappati logicamente agli eventi per la memoria alta e bassa restituita dalla `CreateMemoryResourceNotification` funzione nell'API Windows.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
typedef enum {  
    eMemoryAvailableLow     = 1,  
    eMemoryAvailableNeutral = 2,  
    eMemoryAvailableHigh    = 3
} EMemoryAvailable;  
```  
  
## <a name="members"></a>Membri  
  
|Membro|Description|  
|------------|-----------------|  
|`eMemoryAvailableHigh`|È disponibile molta memoria fisica.|  
|`eMemoryAvailableLow`|È disponibile una quantità di memoria fisica minima.|  
|`eMemoryAvailableNeutral`|La memoria fisica disponibile è neutra.|  
  
## <a name="remarks"></a>Osservazioni  
 Questo valore viene passato dall'host al Common Language Runtime (CLR) utilizzando una chiamata al metodo [ICLRMemoryNotificationCallback:: OnMemoryNotification](iclrmemorynotificationcallback-onmemorynotification-method.md) .
