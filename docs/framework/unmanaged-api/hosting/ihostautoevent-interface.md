# <a name="ihostautoevent-interface"></a>Interfaccia IHostAutoEvent
Fornisce una rappresentazione dell'implementazione dell'host di un evento di reimpostazione automatica.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo Set](ihostautoevent-set-method.md)|Imposta l' `IHostAutoEvent` istanza corrente su uno stato segnalato.|  
|[Metodo Wait](ihostautoevent-wait-method.md)|Fa `IHostAutoEvent` in modo che l'istanza corrente attenda finché l'evento non è di proprietà o non trascorre un intervallo di tempo specificato.|
