# <a name="couninitializecor-function"></a>Funzione CoUninitializeCor
`CoUninitializeCor` è obsoleto.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
STDAPI_(void) CoUninitializeCor(void);  
```  
  
## <a name="remarks"></a>Osservazioni  
 Impossibile scaricare il Common Language Runtime da un processo. Per rimuovere completamente il runtime da un processo in esecuzione, è necessario arrestare il processo.
