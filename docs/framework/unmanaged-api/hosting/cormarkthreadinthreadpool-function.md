# <a name="cormarkthreadinthreadpool-function"></a>Funzione CorMarkThreadInThreadPool
Contrassegna il thread del pool di thread attualmente in esecuzione per l'esecuzione del codice gestito. A partire da .NET Framework versione 2,0, questa funzione non ha alcun effetto. Non è obbligatorio e può essere rimosso dal codice. Questa funzione è deprecata nel .NET Framework 4.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
void CorMarkThreadInThreadPool ();  
```
