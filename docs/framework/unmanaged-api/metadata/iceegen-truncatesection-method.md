# <a name="iceegentruncatesection-method"></a>Metodo ICeeGen::TruncateSection
Tronca la sezione di codice specificata in base alla lunghezza specificata.  
  
 Questo metodo Ã¨ obsoleto e non deve essere utilizzato.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT TruncateSection (  
    [in]  HCEESECTION     section,  
    [in]  ULONG           len  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `section`  
 in Sezione da troncare.  
  
 `len`  
 in Lunghezza, in byte, in base alla quale troncare la sezione.  
  
## <a name="remarks"></a>Commenti  
 Chiamare `TruncateSection` solo se si dispone di requisiti speciali per la sezione che non sono gestiti da altri metodi.
