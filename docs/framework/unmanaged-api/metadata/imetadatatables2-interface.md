# <a name="imetadatatables2-interface"></a>Interfaccia IMetaDataTables2
Estende [IMetaDataTables](imetadatatables-interface.md) per includere metodi per l'utilizzo di flussi di metadati.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo GetMetaDataStorage](imetadatatables2-getmetadatastorage-method.md)|Ottiene le dimensioni e il contenuto dei metadati archiviati nella sezione specificata.|  
|[Metodo GetMetaDataStreamInfo](imetadatatables2-getmetadatastreaminfo-method.md)|Ottiene il nome, le dimensioni e il contenuto del flusso di metadati in corrispondenza dell'indice specificato.|
