# <a name="imetadataemitdefinememberref-method"></a>Metodo IMetaDataEmit::DefineMemberRef
Definisce un riferimento a un membro di un modulo esterno all'ambito corrente e ottiene un token per la definizione di riferimento.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT DefineMemberRef (
    [in]  mdToken           tkImport,
    [in]  LPCWSTR           szName,
    [in]  PCCOR_SIGNATURE   pvSigBlob,
    [in]  ULONG             cbSigBlob,
    [out] mdMemberRef       *pmr
);  
```  
  
## <a name="parameters"></a>Parametri  
 `tkImport`  
 in Token per la classe o l'interfaccia del membro di destinazione, se il membro non è globale; Se il membro è globale, il `mdModuleRef` token per l'altro file.  
  
 `szName`  
 in Nome del membro di destinazione.  
  
 `pvSigBlob`  
 in Firma del membro di destinazione.  
  
 `cbSigBlob`  
 in Numero di byte in `pvSigBlob` .  
  
 `pmr`  
 out `mdMemberRef`Token assegnato.
