# <a name="imetadatavalidate-interface"></a>Interfaccia IMetaDataValidate
Fornisce metodi per convalidare le firme dei metadati.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo ValidateMetaData](imetadatavalidate-validatemetadata-method.md)|Convalida le firme dei metadati degli oggetti nell'ambito dei metadati corrente.|  
|[Metodo ValidatorInit](imetadatavalidate-validatorinit-method.md)|Imposta un flag che specifica il tipo del modulo nell'ambito dei metadati corrente e registra il metodo di callback specificato per gli errori di convalida.|
