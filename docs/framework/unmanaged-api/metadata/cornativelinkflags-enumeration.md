# <a name="cornativelinkflags-enumeration"></a>Enumerazione CorNativeLinkFlags
Fornisce valori di flag usati dal linker durante il collegamento del codice nativo.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
typedef enum  
{  
    nlfNone         = 0x00,  
    nlfLastError    = 0x01,  
    nlfNoMangle     = 0x02,  
    nlfMaxValue     = 0x03  
} CorNativeLinkFlags;  
```  
  
## <a name="members"></a>Membri  
  
|Membro|Descrizione|  
|------------|-----------------|  
|`nlfNone`|Indica nessun flag.|  
|`nlfLastError`|Indica una `setLastError` parola chiave.|  
|`nlfNoMangle`|Indica una `nomangle` parola chiave.|  
|`nlfMaxValue`|Non usato.|
