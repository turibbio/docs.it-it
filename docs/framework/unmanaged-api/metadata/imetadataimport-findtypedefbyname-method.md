# <a name="imetadataimportfindtypedefbyname-method"></a>Metodo IMetaDataImport::FindTypeDefByName
Ottiene un puntatore al token di metadati TypeDef per l'oggetto <xref:System.Type> con il nome specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT FindTypeDefByName  
   [in]  LPCWSTR       szTypeDef,  
   [in]  mdToken       tkEnclosingClass,  
   [out] mdTypeDef     *ptd  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `szTypeDef`  
 in Nome del tipo per il quale ottenere il token TypeDef.  
  
 `tkEnclosingClass`  
 in Token TypeDef o TypeRef che rappresenta la classe contenitore. Se il tipo da trovare non Ã¨ una classe annidata, impostare questo valore su NULL.  
  
 `ptd`  
 out Puntatore al token TypeDef corrispondente.
