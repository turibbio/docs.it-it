# <a name="iclrdatatargetgetmachinetype-method"></a>Metodo ICLRDataTarget::GetMachineType
Ottiene l'identificatore per il tipo di set di istruzioni utilizzato dal processo di destinazione.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetMachineType (  
    [out] ULONG32     *machineType  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `machineType`  
 out Puntatore a un valore che indica il set di istruzioni utilizzato dal processo di destinazione. L'oggetto `machineType` restituito Ã¨ una delle costanti IMAGE_FILE_MACHINE, definite nel file di intestazione Winnt. h.
