# <a name="icordebugthread2getvolatileosthreadid-method"></a>Metodo ICorDebugThread2::GetVolatileOSThreadID
Ottiene l'identificatore del thread del sistema operativo per questo ICorDebugThread2.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetVolatileOSThreadID (  
    [out] DWORD    *pdwTid  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `pdwTid`  
 out Identificatore del thread del sistema operativo per questo thread.
