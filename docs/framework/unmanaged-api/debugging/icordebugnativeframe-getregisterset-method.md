# <a name="icordebugnativeframegetregisterset-method"></a>Metodo ICorDebugNativeFrame::GetRegisterSet
Ottiene il set di registri per questo stack frame.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetRegisterSet (  
    [out] ICorDebugRegisterSet **ppRegisters  
);  
```  
  
## <a name="parameters"></a>Parametri  
 `ppRegisters`  
 out Puntatore all'indirizzo di un oggetto [ICorDebugRegisterSet](icordebugregisterset-interface.md) che rappresenta il set di registri per questo stack frame.
