# <a name="icordebughandlevaluedispose-method"></a>Metodo ICorDebugHandleValue::Dispose
Rilascia l'handle a cui fa riferimento questo oggetto ICorDebugHandleValue senza rilasciare in modo esplicito il puntatore all'interfaccia.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT Dispose ();  
```
