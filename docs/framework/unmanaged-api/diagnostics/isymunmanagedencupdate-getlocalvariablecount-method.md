# <a name="isymunmanagedencupdategetlocalvariablecount-method"></a>Metodo ISymUnmanagedENCUpdate::GetLocalVariableCount
Ottiene il numero di variabili locali.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetLocalVariableCount(  
    [in]  mdMethodDef  mdMethodToken,  
    [out] ULONG        *pcLocals);  
```  
  
## <a name="parameters"></a>Parametri  
 `mdMethodToken`  
 in Token di metadati dei metodi.  
  
 `pcLocals`  
 out Puntatore a un oggetto `ULONG32` che riceve la dimensione, in caratteri, del buffer necessario per contenere il numero di variabili locali.  
  
## <a name="return-value"></a>Valore restituito  
 S_OK se il metodo ha esito positivo; in caso contrario, E_FAIL o un altro codice di errore.  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym. idl, CorSym. h
