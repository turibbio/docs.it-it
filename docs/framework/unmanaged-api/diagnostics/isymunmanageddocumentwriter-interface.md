# <a name="isymunmanageddocumentwriter-interface"></a>Interfaccia ISymUnmanagedDocumentWriter
Fornisce i metodi per la scrittura di un documento cui viene fatto riferimento in un archivio di simboli.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo SetCheckSum](isymunmanageddocumentwriter-setchecksum-method.md)|Imposta le informazioni di checksum.|  
|[Metodo SetSource](isymunmanageddocumentwriter-setsource-method.md)|Imposta l'origine incorporata per un documento in corso di scrittura.|  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym. idl, CorSym. h
