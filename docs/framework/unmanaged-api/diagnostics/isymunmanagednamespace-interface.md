# <a name="isymunmanagednamespace-interface"></a>Interfaccia ISymUnmanagedNamespace
Rappresenta uno spazio dei nomi.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo GetName](isymunmanagednamespace-getname-method.md)|Ottiene il nome di questo spazio dei nomi.|  
|[Metodo GetNamespaces](isymunmanagednamespace-getnamespaces-method.md)|Ottiene gli elementi figlio di questo spazio dei nomi.|  
|[Metodo GetVariables](isymunmanagednamespace-getvariables-method.md)|Restituisce tutte le variabili definite in ambito globale all'interno di questo spazio dei nomi.|  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym. idl, CorSym. h
