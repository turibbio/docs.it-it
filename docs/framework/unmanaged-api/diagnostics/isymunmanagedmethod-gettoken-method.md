# <a name="isymunmanagedmethodgettoken-method"></a>Metodo ISymUnmanagedMethod::GetToken
Restituisce il token di metadati per il metodo.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT GetToken(  
   [out, retval]  mdMethodDef  *pToken);  
```  
  
## <a name="parameters"></a>Parametri  
 `pToken`  
 out Puntatore a un oggetto `mdMethodDef` che riceve la dimensione, in caratteri, del buffer necessario per contenere i metadati.  
  
## <a name="return-value"></a>Valore restituito  
 S_OK se il metodo ha esito positivo; in caso contrario, E_FAIL o un altro codice di errore.  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym. idl, CorSym. h
