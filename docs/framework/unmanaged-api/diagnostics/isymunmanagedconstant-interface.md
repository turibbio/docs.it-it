# <a name="isymunmanagedconstant-interface"></a>Interfaccia ISymUnmanagedConstant
Fornisce l'accesso alle costanti non gestite.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo GetName](isymunmanagedconstant-getname-method.md)|Ottiene il nome della costante.|  
|[Metodo GetSignature](isymunmanagedconstant-getsignature-method.md)|Ottiene la firma della costante.|  
|[Metodo GetValue](isymunmanagedconstant-getvalue-method.md)| Ottiene il valore della costante.|  
  
## <a name="requirements"></a>Requisiti  
 **Intestazione:** CorSym. idl, CorSym. h
