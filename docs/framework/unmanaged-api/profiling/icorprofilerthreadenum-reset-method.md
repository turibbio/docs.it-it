# <a name="icorprofilerthreadenumreset-method"></a>Metodo ICorProfilerThreadEnum::Reset
Sposta il cursore dell'enumeratore nella posizione iniziale della sequenza.  
  
## <a name="syntax"></a>Sintassi  
  
```cpp  
HRESULT Reset();  
```
