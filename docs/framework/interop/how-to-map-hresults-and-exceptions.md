### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="98fb5-114">Per creare una nuova classe di eccezione ed eseguirne il mapping a un HRESULT</span><span class="sxs-lookup"><span data-stu-id="98fb5-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1. <span data-ttu-id="98fb5-115">Usare il codice seguente per creare una nuova classe di eccezione denominata `NoAccessException` ed eseguirne il mapping al valore HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="98fb5-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>  
  
    ```cpp  
    Class NoAccessException : public ApplicationException  
    {  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;
    }  
    }  
    CMyClass::MethodThatThrows  
    {  
    throw new NoAccessException();  
    }  
    ```
