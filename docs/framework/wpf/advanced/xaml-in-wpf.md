# <a name="xaml-in-wpf"></a>XAML in WPF

[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)]è un linguaggio di markup per la programmazione dichiarativa dell'applicazione. [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)]implementa un'implementazione del processore XAML e fornisce il supporto del linguaggio XAML. I [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] tipi vengono implementati in modo che possano fornire il supporto del tipo richiesto per una rappresentazione XAML. In generale, è possibile creare [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] la maggior parte dell'interfaccia utente dell'applicazione nel markup XAML.
