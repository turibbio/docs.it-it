# <a name="exceptionhelper-class"></a>Classe ExceptionHelper

Fornisce le eccezioni con messaggi di errore standardizzati. Questa classe non può essere ereditata.

```csharp
internal static class ExceptionHelper
```

> [!WARNING]
> Questa classe è interna e non è destinata all'uso diretto nel codice.
>
> Microsoft non supporta l'utilizzo di questa classe in un'applicazione di produzione in qualsiasi circostanza.

## <a name="webpermissionunrestricted-field"></a>Campo WebPermissionUnrestricted

Specifica che l'app può connettersi alle risorse Internet.

```csharp
internal static readonly WebPermission WebPermissionUnrestricted
```
