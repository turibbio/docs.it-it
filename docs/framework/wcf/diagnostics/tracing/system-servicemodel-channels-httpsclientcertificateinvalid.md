# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Certificato client non valido.  
  
## <a name="description"></a>Descrizione  
 Questa traccia indica che il certificato fornito dal client è stato ritenuto non valido dal listener HTTPS. Il listener HTTPS ha tentato di convalidare l'autenticità del client usando questo certificato. Un certificato potrebbe non essere valido se l'autorità di certificazione non viene riconosciuta dal server che ospita il servizio.
