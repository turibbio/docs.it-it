# <a name="debugging-on-the-client"></a>Esecuzione del debug sul client
Per semplificare la scrittura di applicazioni client per il servizio WCF da parte degli utenti, è possibile aggiungere il [\<serviceDebug>](../../../configure-apps/file-schema/wcf/servicedebug.md) comportamento del servizio al file di configurazione del servizio. Questo comportamento può essere utilizzato per pubblicare pagine della Guida e per includere fra i dettagli relativi agli errori SOAP restituiti al client le informazioni sulle eccezioni gestite.
