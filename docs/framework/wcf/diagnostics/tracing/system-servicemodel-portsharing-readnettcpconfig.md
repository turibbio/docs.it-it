System.ServiceModel.PortSharing.ReadNetTcpConfig

 I dati di configurazione per NetTcpSection sono stati letti con i dati specificati: listenBacklog, maxConnectionsPendingDispatch, maxPendingAccepts, receiveTimeout e teredoEnabled.
