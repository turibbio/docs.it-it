# <a name="systemservicemodelchannelspeernodeauthenticationtimeout"></a>System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
Handshake di sicurezza non eseguito entro il limite di tempo specificato.  
  
## <a name="description"></a>Descrizione  
 Questa traccia si verifica quando si tenta di stabilire una connessione protetta con un router adiacente.  
  
## <a name="troubleshooting"></a>Risoluzione dei problemi  
 Verificare che tutti i router adiacenti dispongano delle credenziali di sicurezza appropriate e che rispondano a tutti i messaggi in meno di 1 minuto.
