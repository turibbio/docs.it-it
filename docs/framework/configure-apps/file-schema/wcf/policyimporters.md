# \<policyImporters>
Questo elemento di configurazione specifica tutte le unità di importazione dei criteri che controllano l'importazione di asserzioni di criteri personalizzate relative alle associazioni. Ogni elemento figlio è un <`policyImporter`> utilizzato per la ricerca di asserzioni di criteri personalizzate sulle funzionalità di associazione, nonché per il collegamento di un elemento di associazione personalizzato che implementa le funzionalità richieste dall'asserzione.
