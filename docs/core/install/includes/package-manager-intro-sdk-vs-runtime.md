Installare l'SDK (che include il runtime) se si desidera sviluppare app .NET Core.Install the SDK (which includes the runtime) if you want to develop .NET Core apps. In alternativa, se è sufficiente eseguire le app, installare il runtime. Se si sta installando il runtime, si consiglia di installare il [runtime di ASP.NET Core](#install-the-aspnet-core-runtime), in quanto include runtime .NET Core e ASP.NET Core.

Se l'SDK o il runtime è `dotnet --list-sdks` `dotnet --list-runtimes` già stato installato, utilizzare i comandi e per vedere quali versioni sono installate. Per ulteriori informazioni, vedere [Come verificare che .NET Core sia già installato.](../how-to-detect-installed-versions.md?pivots=os-linux)
