# <a name="tutorial-manipulating-content-in-a-wordprocessingml-document-visual-basic"></a>Esercitazione: modifica del contenuto in un documento WordprocessingML (Visual Basic)

 Per ulteriori informazioni, vedere il [Blog di Eric White](http://www.ericwhite.com).
