# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Assi integrati nel linguaggio in Visual Basic (LINQ to XML)
In questa sezione vengono descritte le funzionalità integrate direttamente nel linguaggio Visual Basic per semplificare l'accesso a XML. Molti degli esempi della documentazione LINQ to XML utilizzano questi assi Visual Basic integrati.  
  
## <a name="in-this-section"></a>Contenuto della sezione  
  
|Argomento|Descrizione|  
|-----------|-----------------|  
|[XML Child Axis Property](../../../language-reference/xml-axis/xml-child-axis-property.md)|Fornisce l'accesso agli elementi figlio di uno dei seguenti oggetti: <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, raccolta di <xref:System.Xml.Linq.XElement> o raccolta di <xref:System.Xml.Linq.XDocument>. Questo asse è equivalente all'asse <xref:System.Xml.Linq.XContainer.Elements%2A>.|  
|[XML Descendant Axis Property](../../../language-reference/xml-axis/xml-descendant-axis-property.md)|Fornisce l'accesso ai discendenti di uno dei seguenti oggetti: <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument> o raccolta di <xref:System.Xml.Linq.XElement>. Questo asse è equivalente all'asse <xref:System.Xml.Linq.XContainer.Descendants%2A>.|  
|[XML Attribute Axis Property](../../../language-reference/xml-axis/xml-attribute-axis-property.md)|Fornisce l'accesso a un attributo di un oggetto <xref:System.Xml.Linq.XElement>. Questo asse è quasi equivalente all'asse <xref:System.Xml.Linq.XElement.Attribute%2A>. Questo asse è diverso dall'asse <xref:System.Xml.Linq.XElement.Attribute%2A>, in quanto restituisce il valore dell'attributo, non un oggetto <xref:System.Xml.Linq.XAttribute>.|  
|[Proprietà dell'indicizzatore di estensione](../../../language-reference/xml-axis/extension-indexer-property.md)|Fornisce l'accesso ai singoli elementi di una raccolta.|
