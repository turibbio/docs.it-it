# <a name="exit-operator-is-not-valid-use-return-to-exit-an-operator"></a>'Exit Operator' non è valido. Usare 'Return' per uscire da un operatore
Un'istruzione `Exit Operator` viene visualizzata in una routine `Operator` .  
  
 Per restituire una routine, è necessario utilizzare un' [istruzione return](../language-reference/statements/return-statement.md) `Operator` . L' [istruzione Exit](../language-reference/statements/exit-statement.md) non accetta la `Operator` parola chiave e l' `End Operator` istruzione non restituisce il controllo al codice chiamante.  
  
 **ID errore:** BC33008  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Sostituire l'istruzione `Exit Operator` con un'istruzione `Return` .
