# <a name="field-fieldname-is-of-an-unsupported-type"></a>Il campo '\<fieldname>' è di tipo non supportato
Un campo viene dichiarato come un tipo che non può essere rappresentato da Visual Studio.  
  
 **ID errore:** BC30656  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Definire il campo come un tipo supportato.  
  
2. Sviluppare un nuovo tipo che descrive i dati nel campo.
