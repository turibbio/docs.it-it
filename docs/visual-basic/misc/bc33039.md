 <span data-ttu-id="2090a-104">La variabile contatore deve essere di un tipo che supporta operatori di aggiunta (`+`) e sottrazione (`-`) che agiscono completamente nel relativo tipo contenitore.</span><span class="sxs-lookup"><span data-stu-id="2090a-104">The counter variable must be of a type that supports addition (`+`) and subtraction (`-`) operators that operate completely on their containing type.</span></span> <span data-ttu-id="2090a-105">Ciò significa che sia gli operandi che il valore restituito devono essere del tipo della variabile contatore.</span><span class="sxs-lookup"><span data-stu-id="2090a-105">This means both of the operands and the return value must be of the type of the counter variable.</span></span>
