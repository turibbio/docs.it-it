# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>' \<membername1> ' non può eseguire l'override \<membername2> di '' perché si differenziano per i tipi di parametri facoltativi
I membri di overload devono presentare tipi di dati diversi per uno o più parametri obbligatori.  
  
 **ID errore:** BC30697  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Definire almeno un parametro obbligatorio come tipo di dati diverso.
