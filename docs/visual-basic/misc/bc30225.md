# <a name="option-specifier-statement-can-only-appear-once-per-file"></a>L'istruzione dell'opzione \<specifier>' può essere contenuta una sola volta per file
Le istruzioni `Option Compare`, `Option Explicit`e `Option Strict` possono essere usate una sola volta nello stesso file di origine.  
  
 **ID errore:** BC30225  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere l'istruzione `Option` duplicata.
