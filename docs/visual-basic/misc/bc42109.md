# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>La variabile '\<variablename>' è stata usata prima dell'assegnazione di un valore (errore di Visual Basic)
Una variabile è stata usata prima dell'assegnazione di un valore, con la conseguente generazione di un'eccezione <xref:System.NullReferenceException>.  
  
 **ID errore:** BC42109  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che alla variabile sia stato assegnato un valore.
