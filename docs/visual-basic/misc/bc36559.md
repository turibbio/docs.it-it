# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="0d519-102">Impossibile usare la proprietà di membro di tipo anonimo '\<propertyname>' per dedurre il tipo di un'altra proprietà di membro perché il tipo di '\<propertyname>' non è ancora stabilito</span><span class="sxs-lookup"><span data-stu-id="0d519-102">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="0d519-103">Fino a quando non viene stabilito il tipo di una proprietà di tipo anonimo, non può essere usata per stabilire il tipo di un'altra proprietà.</span><span class="sxs-lookup"><span data-stu-id="0d519-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="0d519-104">Ad esempio, nella dichiarazione seguente `.IDName = .LastName` non è valido perché `.LastName` non è ancora stato inizializzato.</span><span class="sxs-lookup"><span data-stu-id="0d519-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.
Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}
```
