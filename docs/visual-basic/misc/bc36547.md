# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="4e025-102">Proprietà o membro di tipo anonimo '\<propertyname>' già dichiarato</span><span class="sxs-lookup"><span data-stu-id="4e025-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="4e025-103">Un nome di proprietà può essere definito una sola volta nella dichiarazione di un tipo anonimo.</span><span class="sxs-lookup"><span data-stu-id="4e025-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="4e025-104">Ad esempio, le dichiarazioni seguenti non sono valide:</span><span class="sxs-lookup"><span data-stu-id="4e025-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="4e025-105">**ID errore:** BC36547</span><span class="sxs-lookup"><span data-stu-id="4e025-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4e025-106">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="4e025-106">To correct this error</span></span>  
  
- <span data-ttu-id="4e025-107">Scegliere un nome diverso per una delle proprietà.</span><span class="sxs-lookup"><span data-stu-id="4e025-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.

```

```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
