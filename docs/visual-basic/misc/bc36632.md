# <a name="nullable-parameters-must-specify-a-type"></a>I parametri nullable devono specificare un tipo
Un parametro include un modificatore di tipo nullable (?) ma non specifica un tipo.  
  
 **ID errore:** BC36632  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Specificare un tipo quando si dichiara il parametro di tipo valore nullable.
