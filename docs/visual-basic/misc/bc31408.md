# <a name="specifier1-and-specifier2-cannot-be-combined"></a>' \<specifier1> ' è \<specifier2> ' non possono essere combinati
Due identificatori incompatibili, ad esempio `Shadows` e `Overrides` , vengono usati insieme.  
  
 **ID errore:** BC31408  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere uno degli identificatori.
