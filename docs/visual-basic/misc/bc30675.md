# <a name="synclock-statement-must-end-with-a-matching-end-synclock"></a>L'istruzione 'SyncLock' deve terminare con un oggetto 'End SyncLock' corrispondente
`SyncLock` iniziano con la parola chiave `SyncLock` e terminano con il costrutto `End SyncLock` .  
  
 **ID errore:** BC30675  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che il blocco `SyncLock` termini con un costrutto `End SyncLock` .
