# <a name="labels-are-not-valid-in-the-immediate-window"></a>Etichette non valide nella finestra di controllo immediato
Le etichette delle righe abilitano la diramazione e non sono consentite in un contesto di debug.  
  
 **ID errore:** BC30134  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Non immettere un'etichetta della riga nella finestra di controllo **immediato** .
