# <a name="expected-closing--for-xml-entity"></a>Previsto ';' di chiusura per l'entità XML
In un riferimento a entità XML (ad esempio, `>`) manca un punto e virgola di chiusura (;).  
  
 **ID errore:** BC31178  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Includere il punto e virgola alla fine del riferimento a entità XML.
