# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>'End RaiseEvent' deve essere la prima istruzione in una riga
Un'istruzione `End RaiseEvent` segue un separatore di istruzioni due punti (:). `End RaiseEvent` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC31120
