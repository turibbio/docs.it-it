# <a name="property-parameters-cannot-be-declared-byref"></a>\<Property>I parametri  non possono essere dichiarati 'ByRef'
I parametri per le proprietà devono essere dichiarati usando il modificatore `ByVal` .  
  
 **ID errore:** BC30651  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Definire i parametri di proprietà come `ByVal`.
