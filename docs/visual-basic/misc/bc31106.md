# <a name="property-accessors-cannot-be-declared-accessmodifier-in-a-notoverridable-property"></a><span data-ttu-id="e07db-102">Le funzioni di accesso alle proprietà non possono essere dichiarate '\<accessmodifier>' in una proprietà 'NotOverridable'</span><span class="sxs-lookup"><span data-stu-id="e07db-102">Property accessors cannot be declared '\<accessmodifier>' in a 'NotOverridable' property</span></span>
<span data-ttu-id="e07db-103">Un' [istruzione Get](../language-reference/statements/get-statement.md) o un' [istruzione set](../language-reference/statements/set-statement.md) in una `NotOverridable` proprietà include la `Private` parola chiave.</span><span class="sxs-lookup"><span data-stu-id="e07db-103">A [Get Statement](../language-reference/statements/get-statement.md) or [Set Statement](../language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="e07db-104">Di seguito viene spiegato perché `NotOverridable` e `Private` non possono essere combinate in una [Property Statement](../language-reference/statements/property-statement.md):</span><span class="sxs-lookup"><span data-stu-id="e07db-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../language-reference/statements/property-statement.md):</span></span>  
  
1. <span data-ttu-id="e07db-105">Una proprietà o una routine che non esegue l'override di una proprietà o di una routine di classe base è caratterizzata dall'impostazione predefinita [NotOverridable](../language-reference/modifiers/notoverridable.md).</span><span class="sxs-lookup"><span data-stu-id="e07db-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../language-reference/modifiers/notoverridable.md).</span></span>  
  
2. <span data-ttu-id="e07db-106">Tuttavia, una proprietà o una routine di una classe derivata che esegue l'override di una proprietà o di una routine di classe base è caratterizzata dall'impostazione predefinita [Overridable](../language-reference/modifiers/overridable.md).</span><span class="sxs-lookup"><span data-stu-id="e07db-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="e07db-107">Per terminare la gerarchia di override, è possibile dichiararla `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="e07db-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="e07db-108">Questo è il solo contesto in cui si può usare la proprietà `NotOverridable`.</span><span class="sxs-lookup"><span data-stu-id="e07db-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="e07db-109">In altre parole, `NotOverridable` può essere usata soltanto in combinazione con [Overrides](../language-reference/modifiers/overrides.md).</span><span class="sxs-lookup"><span data-stu-id="e07db-109">That is, you can use `NotOverridable` only in combination with [Overrides](../language-reference/modifiers/overrides.md).</span></span>  
  
3. <span data-ttu-id="e07db-110">Se una proprietà o una routine di classe base viene dichiarata [Private](../language-reference/modifiers/private.md), una classe derivata non può eseguire l'override di quella proprietà o di quella routine in quanto non può accedervi.</span><span class="sxs-lookup"><span data-stu-id="e07db-110">If a base class property or procedure is declared [Private](../language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="e07db-111">Per questo motivo, non è possibile usare `Private` in combinazione con `Overridable`.</span><span class="sxs-lookup"><span data-stu-id="e07db-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4. <span data-ttu-id="e07db-112">Per poter eseguire l'override di una proprietà o di una routine, la proprietà o la routine che lo esegue deve avere in comune con queste non solo una firma identica, ma anche il medesimo livello di accesso.</span><span class="sxs-lookup"><span data-stu-id="e07db-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="e07db-113">Questo implica che una proprietà o una routine che esegue un override non può essere `Private`perché una proprietà o una routine di cui può essere eseguito l'override non può essere dichiarata `Private`.</span><span class="sxs-lookup"><span data-stu-id="e07db-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5. <span data-ttu-id="e07db-114">Quindi, poiché la proprietà `NotOverridable` può essere dichiarata solo per una proprietà o una routine che esegue l'override, non è possibile combinarla con la parola chiave `Private`.</span><span class="sxs-lookup"><span data-stu-id="e07db-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="e07db-115">In base allo stesso ragionamento, le routine di proprietà individuali, vale a dire`Get` e `Set`, di una proprietà che esegue l'override non possono essere `Private`.</span><span class="sxs-lookup"><span data-stu-id="e07db-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>
