# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Il tipo '\<typename>' non può essere usato come attributo perché non è dichiarato come 'Public'
Gli attributi devono essere pubblici.  
  
 **ID errore:** BC31516  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Aggiungere il modificatore `Public` alla classe Attribute.
