# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="efa0c-102">Non è possibile dedurre da questi argomenti i tipi di dati dei parametri di tipo nel metodo di estensione '\<methodname>' definito in 'typename' perché non vengono convertiti nello stesso tipo</span><span class="sxs-lookup"><span data-stu-id="efa0c-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type</span></span>

<span data-ttu-id="efa0c-103">Non è possibile dedurre da questi argomenti i tipi di dati dei parametri di tipo nel metodo di estensione ' \<methodname> ' definito in ' typename ' perché non vengono convertiti nello stesso tipo.</span><span class="sxs-lookup"><span data-stu-id="efa0c-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="efa0c-104">Per correggere l'errore, provare a specificare i tipi di dati in modo esplicito.</span><span class="sxs-lookup"><span data-stu-id="efa0c-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="efa0c-105">Si è provato a usare l'inferenza del tipo per determinare il tipo (o i tipi) di dati del parametro (o dei parametri) di tipo durante la valutazione di una chiamata a un metodo di estensione generico.</span><span class="sxs-lookup"><span data-stu-id="efa0c-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="efa0c-106">Il compilatore non ha trovato un tipo di dati che soddisfi i vincoli di tutti gli argomenti.</span><span class="sxs-lookup"><span data-stu-id="efa0c-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="efa0c-107">Di conseguenza, ha segnalato questo errore.</span><span class="sxs-lookup"><span data-stu-id="efa0c-107">Therefore, it reported this error.</span></span>

> [!NOTE]
> <span data-ttu-id="efa0c-108">Quando non è possibile specificare gli argomenti (ad esempio per gli operatori di query nelle espressioni di query), il messaggio di errore visualizzato non contiene la seconda frase.</span><span class="sxs-lookup"><span data-stu-id="efa0c-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="efa0c-109">Il codice seguente illustra l'errore.</span><span class="sxs-lookup"><span data-stu-id="efa0c-109">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Module Module3
    Sub Main()

        Dim c1 As New Class1

        '' Not valid. Integer and Date do not convert to the same type.
        'c1.targetMethod(19, #3/4/2007#)

    End Sub

    <System.Runtime.CompilerServices.Extension()> _
    Sub targetMethod(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```

## <a name="to-correct-this-error"></a><span data-ttu-id="efa0c-111">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="efa0c-111">To correct this error</span></span>

