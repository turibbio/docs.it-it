# <a name="xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement"></a>Il parametro di tipo \<parameter> del commento XML non corrisponde a un parametro di tipo nell'istruzione \<keyword> corrispondente
Il parametro del commento XML non \<parameter> corrisponde a un parametro di tipo nell' \<keyword> istruzione corrispondente. Il commento XML verrà ignorato.  
  
 Un tag di commenti XML è stato specificato per alcuni, ma non per tutti, i parametri di tipi in un metodo.  
  
 **ID errore:** BC42317  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Specificare i tag di commento XML per tutti i parametri di tipo nel metodo.
