# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>Risoluzione dell'overload non riuscita perché nessun '\<method>' accessibile può essere chiamato con questi argomenti: \<error>
È stata effettuata una chiamata a un metodo di overload, ma il compilatore non riesce a trovare un metodo con un elenco di argomenti corrispondenti.  
  
 **ID errore:** BC30518  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Controllare l'elenco di argomenti fornito.
