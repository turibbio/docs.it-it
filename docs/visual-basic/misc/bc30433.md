# <a name="methods-in-a-module-cannot-be-declared-specifier"></a>I metodi in un modulo non possono essere dichiarati ' \<specifier> '
È stato usato un identificatore non valido in un metodo all'interno di un'istruzione `Module` . Non è possibile creare istanze dei moduli, i moduli non supportano l'ereditarietà e non possono implementare le interfacce.  
  
 **ID errore:** BC30433  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere l'identificatore.
