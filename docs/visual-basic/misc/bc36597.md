# <a name="goto-to-label-labelname-cannot-be-used-within-lambda-expressions"></a>Non è possibile usare un'istruzione GoTo all'interno delle espressioni lambda per andare all'etichetta '\<labelname>'
Il corpo di un'espressione lambda deve essere un'espressione che calcola e restituisce un singolo valore. Non sono consentite istruzioni, comprese le istruzioni `GoTo` .  
  
 **ID errore:** BC36597
