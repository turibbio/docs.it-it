# <a name="argument-argumentname-cannot-be-converted-to-type-typename"></a>Impossibile convertire l'argomento ' \<argumentname> ' nel tipo ' \<typename> '
Una conversione tenta di modificare un argomento a un tipo non valido.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Modificare il tipo in un tipo valido.
