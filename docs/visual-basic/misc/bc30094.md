# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>L'etichetta ' \<labelname> ' è già definita nel metodo o nell'espressione lambda su più righe corrente
Il nome dell'etichetta viene duplicato all'interno della routine. Tutti i nomi all'interno di una routine devono essere univoci.  
  
 **ID errore:** BC30094  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Rinominare l'etichetta duplicata.
