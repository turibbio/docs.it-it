# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>Le classi gestite derivate da una classe COM non possono essere chiamate ad associazione tardiva.

Si è provato a eseguire una chiamata ad associazione tardiva a una classe gestita derivata da una classe COM, ma questo tipo di chiamata non è supportato.

## <a name="to-correct-the-problem"></a>Per risolvere il problema

Eseguire una chiamata con associazione anticipata.
