# <a name="addressof-expressions-are-not-valid-in-debug-windows"></a>Le espressioni 'AddressOf' non sono valide nelle finestre di debug
Le istruzioni`AddressOf` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30731  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Rimuovere le istruzioni `AddressOf` dal codice di debug.
