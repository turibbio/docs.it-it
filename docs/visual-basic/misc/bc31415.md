# <a name="implementsclause-cannot-implement-typename-because-typename-is-a-reserved-name"></a>' \<implementsclause> ' non può implementare '' \<typename> perché' \<typename> ' è un nome riservato
Un'istruzione `Implements` ha specificato un nome riservato.  
  
 **ID errore:** BC31415  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rinominare il membro di interfaccia che non può essere implementato.
