# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="81ea2-102">' \<derivedtypename> ' non può ereditare da \<type> ' \<constructedbasetypename> ' perché espande l'accesso di tipo ' \<internaltypename> ' a \<region> ' \<regionname> '</span><span class="sxs-lookup"><span data-stu-id="81ea2-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="81ea2-103">Una classe o un'interfaccia derivata prova a espandere il livello di accesso di un tipo interno usandolo come argomento di tipo per una classe o un'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="81ea2-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="81ea2-104">Il codice seguente può generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="81ea2-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="81ea2-105">Al codice esterno a `containingClass` non è consentito l'accesso a `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="81ea2-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="81ea2-106">Tuttavia, è possibile accedere a `derivedClass` da qualsiasi codice nello stesso assembly.</span><span class="sxs-lookup"><span data-stu-id="81ea2-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="81ea2-107">Per questo motivo `derivedClass` non può ereditare `baseClass` se usa `internalStructure` come argomento di tipo, perché ciò potrebbe esporre `internalStructure` in tutta la regione del codice di definizione.</span><span class="sxs-lookup"><span data-stu-id="81ea2-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="81ea2-108">**ID errore:** BC30921</span><span class="sxs-lookup"><span data-stu-id="81ea2-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="81ea2-109">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="81ea2-109">To correct this error</span></span>  
  
- <span data-ttu-id="81ea2-110">Regolare i livelli di accesso delle classi o delle interfacce in modo che il tipo derivato non espanda il livello di accesso del tipo interno.</span><span class="sxs-lookup"><span data-stu-id="81ea2-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="81ea2-111">-oppure-</span><span class="sxs-lookup"><span data-stu-id="81ea2-111">-or-</span></span>  
  
- <span data-ttu-id="81ea2-112">Se non è possibile regolare i livelli di accesso, non usare il tipo interno come argomento di tipo durante la costruzione della classe o dell'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="81ea2-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
