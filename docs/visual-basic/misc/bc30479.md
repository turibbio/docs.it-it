# <a name="shared-sub-new-cannot-have-any-parameters"></a>'Sub New' condiviso non può accettare parametri
Uno o più parametri sono stati inclusi in un'istruzione `Sub New` condivisa. Questa operazione non è valida.  
  
 **ID errore:** BC30479  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere il parametro o i parametri.
