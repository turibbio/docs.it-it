# <a name="enum-types-cannot-be-nullable"></a>I tipi enum non possono essere nullable
Il tipo sottostante usato per dichiarare un'enumerazione non pu√≤ essere un tipo di valore Nullable. Il codice seguente, ad esempio, causa questo errore:  
  
