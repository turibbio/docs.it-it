# <a name="a-nullable-type-cannot-be-inferred-for-variable-variablename"></a>Un tipo nullable non può essere\<dedotto per la variabile ' nomevariabile>'
Un tipo di valore nullable non può essere dedotto da un `String`tipo di riferimento, ad esempio una matrice, una classe o un oggetto . Il valore da cui viene dedotto il tipo di dati deve essere un tipo valore. Questo errore è illustrato nel codice seguente.  
  
```vb  
'' Not valid.
'Dim arrList? = New ArrayList  
'Dim except? = New Exception  
'Dim obj? = New Object  
'Dim stringVar? = "Open the application."  
  
' Valid.  
Dim intVar? = 10  
```  
  
 **ID errore:** BC36628  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere la designazione nullable.
