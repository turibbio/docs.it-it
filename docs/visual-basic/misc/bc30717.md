# <a name="try-statements-are-not-valid-in-the-immediate-window"></a>Le istruzioni 'Try' non sono valide nella finestra di controllo immediato
Le istruzioni`Try` possono essere usate solo nel codice sorgente.  
  
 **ID errore:** BC30717  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere le istruzioni `Try` dal codice di debug.
