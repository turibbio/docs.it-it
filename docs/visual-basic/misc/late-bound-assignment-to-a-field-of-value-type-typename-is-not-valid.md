# <a name="late-bound-assignment-to-a-field-of-value-type-typename-is-not-valid-when-name-is-the-result-of-a-late-bound-expression"></a>Assegnazione ad associazione tardiva a un campo di tipo di valore '\<typename>' non valida quando il risultato di un'espressione ad associazione tardiva è '\<name>'
Si è tentata un'assegnazione ad associazione tardiva non valida.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Eseguire l'assegnazione con associazione anticipata.
