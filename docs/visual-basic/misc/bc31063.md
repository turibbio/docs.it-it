# <a name="set-method-cannot-have-more-than-one-parameter"></a>Il metodo 'Set' non può avere più di un parametro
Più parametri sono stati definiti per le routine di una proprietà `Set` .  
  
 **ID errore:** BC31063  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere i parametri aggiuntivi dalle routine `Set` di una proprietà.
