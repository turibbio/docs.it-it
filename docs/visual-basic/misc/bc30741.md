# <a name="typename1-cannot-be-converted-to-typename2"></a>' \<typename1> ' non può essere convertito in ' \<typename2> '
È stata tentata una conversione del tipo tra due tipi incompatibili.  
  
 **ID errore:** BC30741  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare l'esecuzione del cast dei tipi.
