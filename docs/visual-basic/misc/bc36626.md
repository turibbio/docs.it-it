# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>Impossibile convertire l'espressione Lambda in '\<typename>' perché il tipo '\<typename>' è dichiarato come 'MustInherit' ed è impossibile crearlo
Un'istruzione tenta di convertire un'espressione lambda in un tipo che può essere solo una classe base e non può essere usato per creare un'istanza.  
  
 **ID errore:** BC36626
