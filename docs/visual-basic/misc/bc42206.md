# <a name="maximum-number-of-warnings-has-been-exceeded"></a>Superato il numero massimo di avvisi consentiti
Il numero massimo di avvisi di compilazione consentiti è 100. Il compilatore impone questo limite per ridurre le dimensioni dei report di errore e avviso.  
  
 Per impostazione predefinita, si tratta di un messaggio di avviso. Per altre informazioni su come nascondere gli avvisi o considerarli come errori, vedere [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID errore:** BC42206  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Esaminare gli altri avvisi generati dalla compilazione e risolvere il maggior numero possibile di tali avvisi. Se il codice sorgente contiene più di 100 avvisi, è possibile visualizzare altri avvisi nella compilazione successiva.
