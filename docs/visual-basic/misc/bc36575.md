# <a name="anonymous-type-member-name-must-be-preceded-by-a-period"></a>Il nome di membro di tipo anonimo deve essere preceduto da un punto
Nell'elenco degli inizializzatori di oggetto per una dichiarazione di tipo anonimo, un nuovo nome di membro a cui Ã¨ assegnato un valore deve essere preceduto da un punto. L'esempio seguente mostra una dichiarazione valida e una dichiarazione non valida:  
  
```vb  
' Valid.  
Dim instanceName1 = New With {.memberName = 10}  
' Invalid declaration that causes this error.  
' Dim instanceName2 = New With {memberName = 10}  
```  
  
 **ID errore:** BC36575  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Aggiungere un punto prima del nome del membro.
