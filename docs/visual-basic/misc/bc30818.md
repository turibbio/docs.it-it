# <a name="function-is-not-declared"></a>' \<function> ' non è dichiarato
'\<function>' non è dichiarato. La funzione è stata spostata nello spazio dei nomi Microsoft.VisualBasic.  
  
 **ID errore:** BC30818  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Usare la funzione appropriata dallo spazio dei nomi `Microsoft.VisualBasic` .
