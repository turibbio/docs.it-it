# <a name="type-characters-are-not-allowed-in-label-identifiers"></a>I caratteri tipo non sono consentiti negli identificatori di etichetta
Un'etichetta di un'istruzione contiene almeno un carattere di tipo identificatore.  
  
 Un'etichetta di istruzione deve essere un nome Visual Basic valido. I caratteri consentiti non includono i caratteri di tipo identificatore, ovvero`%`, `&`, `@`, `!`, `#`e `$`. Vedere [Declared Element Names](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID errore:** BC31395  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere il carattere o i caratteri di tipo identificatore dall'etichetta di istruzione.
