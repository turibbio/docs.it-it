# <a name="automation-object-does-not-have-a-default-value"></a>L'oggetto di automazione non ha un valore predefinito
Visual Basic non è in grado di determinare il membro predefinito per l'oggetto specificato.  
  
### <a name="to-correct-the-error"></a>Per correggere l'errore  
  
- Controllare la documentazione dell'oggetto e fornire una specifica esplicita per la proprietà o il metodo.
