# <a name="duplicate-xml-attribute-attributename"></a>L'attributo XML 'attributeName' è duplicato
Un attributo specificato per un valore letterale elemento XML corrisponde a quello di un attributo che è stato definito in precedenza per il valore letterale elemento XML. I nomi degli attributi in valori letterali dell'elemento XML devono essere univoci.  
  
 **ID errore:** BC31149  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rinominare l'attributo con un nome univoco per il valore letterale elemento XML.
