# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>Il tag 'include' del commento XML deve avere un attributo '\<attribute>'
Il tag ' include ' del commento XML deve avere un \<attribute> attributo. Il commento XML verrà ignorato.  
  
 Un attributo `file` o `path` , passato al tag `<`include`>` , è assente o incompleto.  
  
 **ID errore:** BC42310  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Aggiungere o completare l'attributo obbligatorio per il \<include> tag.
