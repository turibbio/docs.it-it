# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>Il tag '\<tag>' del commento XML è presente più volte con attributi identici nello stesso blocco di commento XML
Il tag del commento XML è \<tag> presente più volte con attributi identici nello stesso blocco di commento XML. Il commento XML verrà ignorato.  
  
 Il commento XML contiene più tag con attributi identici.  
  
 **ID errore:** BC42305  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere i tag duplicati.
