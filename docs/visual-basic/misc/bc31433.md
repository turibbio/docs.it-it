# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Il metodo '\<methodname>' non può essere dichiarato come 'Partial' perché solo un metodo '\<methodname>' può essere contrassegnato come 'Partial'
È possibile avere due dichiarazioni di metodo parziale con lo stesso nome e firma.  
  
 **ID errore:** BC31433  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Modificare il nome o la firma di una delle dichiarazioni di metodo parziale.
