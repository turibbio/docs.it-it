# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'New' non può essere usato nella classe '\<classname>' perché contiene un membro 'MustOverride' di cui non è stato eseguito l'override
Si è provato a usare `New` in una classe che contiene un membro `MustOverride` non sottoposto a override.  
  
 **ID errore:** BC30376  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere l'istruzione `New` .
