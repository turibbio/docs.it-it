# <a name="class-in-an-interface-cannot-be-declared-specifier"></a>La classe in un'interfaccia non può essere dichiarata come ' \<specifier> '
Il modificatore specificato con una classe dichiarata in un'interfaccia non è valido.  
  
 **ID errore:** BC31070  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere i modificatori come `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`o `Overridable`.
