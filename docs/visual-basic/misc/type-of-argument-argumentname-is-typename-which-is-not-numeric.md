# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>Il tipo dell'argomento '\<argumentname>' è '\<typename>'. Tale tipo non è numerico
Un argomento non numerico si verifica in una situazione in cui è previsto un argomento numerico.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che l'argomento sia numerico.
