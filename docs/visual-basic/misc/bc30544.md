# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>Il metodo non può contenere un'istruzione 'Try' insieme a un'istruzione 'On Error' o 'Resume'
È stata combinata un'istruzione `Try` con `On Error` o `Resume`.  
  
 **ID errore:** BC30544  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Rimuovere `On Error` o `Resume`.
