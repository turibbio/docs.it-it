# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>Il parametro '\<parameter>' del commento XML non corrisponde a un parametro nell'istruzione \<keyword> corrispondente
Il parametro del commento XML non \<parameter> corrisponde a un parametro nell' \<keyword> istruzione corrispondente. Il commento XML verrà ignorato.  
  
 Un tag di commenti XML è stato specificato per alcuni, ma non per tutti, i parametri in un metodo.  
  
 **ID errore:** BC42307  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Specificare il tag di commento XML per tutti i parametri nel metodo.
