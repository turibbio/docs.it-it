# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>' \<membername1> ' è \<membername2> ' non possono essere in overload perché si differenziano solo per i tipi di parametri facoltativi
I membri di overload devono presentare tipi di dati diversi per uno o più parametri obbligatori.  
  
 **ID errore:** BC30696  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Definire almeno un parametro obbligatorio come tipo di dati diverso.
