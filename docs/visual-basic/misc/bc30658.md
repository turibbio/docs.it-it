# <a name="property-propertyname-with-no-parameters-cannot-be-found"></a>Impossibile trovare la proprietà '\<propertyname>' senza parametri
L'attributo usato ha una proprietà che richiede uno o più parametri.  
  
 **ID errore:** BC30658  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Specificare i parametri per la proprietà.
