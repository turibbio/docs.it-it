# <a name="nullable-types-are-not-supported-because-the-current-version-of-mscorlibdll-does-not-define-the-type-nullableof-t"></a>I tipi nullable non sono supportati perché la versione corrente di 'mscorlib.dll' non definisce il tipo 'Nullable(of T)'
La versione del Framework in uso non supporta i tipi di valore Nullable.  
  
 **ID errore:** BC32130
