# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>'End SyncLock' deve essere preceduto da un oggetto 'SyncLock' corrispondente
`SyncLock` iniziano con la parola chiave `SyncLock` e terminano con il costrutto `End SyncLock` .  
  
 **ID errore:** BC30674  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che il blocco `SyncLock` inizi con un'istruzione `SyncLock` .
