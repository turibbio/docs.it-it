# <a name="mycomputerregistry-object"></a>Oggetto My.Computer.Registry
Specifica proprietà e metodi per la modifica del Registro di sistema.  
  
## <a name="remarks"></a>Commenti  
 Per informazioni sui metodi e le proprietà dell'oggetto `My.Computer.Registry` , vedere <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.  
  
 Per ulteriori informazioni, vedere [lettura e scrittura nel registro di sistema](../../developing-apps/programming/computer-resources/reading-from-and-writing-to-the-registry.md).  
  
> [!NOTE]
> È anche possibile modificare il registro di sistema usando i metodi della <xref:Microsoft.Win32.Registry?displayProperty=nameWithType> classe.  
  
## <a name="requirements"></a>Requisiti  
 **Spazio dei nomi:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Classe:** <xref:Microsoft.VisualBasic.MyServices.RegistryProxy> (fornisce l'accesso a <xref:Microsoft.Win32.Registry> )  
  
 **Assembly:** Libreria di runtime Visual Basic (in Microsoft. VisualBasic. dll)
