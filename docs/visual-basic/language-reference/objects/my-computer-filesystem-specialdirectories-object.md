# <a name="mycomputerfilesystemspecialdirectories-object"></a>Oggetto My.Computer.FileSystem.SpecialDirectories
Specifica le proprietà per l'accesso alle directory a cui si fa comunemente riferimento.  
  
## <a name="remarks"></a>Commenti  
 Per informazioni sui metodi e le proprietà dell'oggetto `My.Computer.FileSystem.SpecialDirectories` , vedere <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 Per ulteriori informazioni, vedere [procedura: recuperare il contenuto della directory documenti](../../developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).  
  
## <a name="requirements"></a>Requisiti  
 **Spazio dei nomi:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Classe:** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy> (fornisce l'accesso a <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories> )  
  
 **Assembly:** Libreria di runtime Visual Basic (in Microsoft. VisualBasic. dll)
