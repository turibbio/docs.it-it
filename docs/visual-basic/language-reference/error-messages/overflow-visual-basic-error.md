# <a name="overflow-visual-basic-error"></a>Overflow (errore Visual Basic)
Un valore letterale rappresenta un valore che non rientra nei limiti del tipo di dati a cui viene assegnato.  
  
 **ID errore:** BC30036  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Consultare l'intervallo di valori per il tipo di dati di destinazione e riscrivere il valore letterale per conformarsi a tale intervallo.
