# <a name="property-array-index-is-not-valid"></a>Indice della matrice di proprietà non valido
Il valore fornito non è valido per l'indice di una matrice di proprietà.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Controllare la documentazione del componente per assicurarsi che l'indice sia compreso nell'intervallo valido per la proprietà specificata.
