# <a name="device-io-error"></a>Errore di I/O del dispositivo
Si è verificato un errore di input o di output durante il programma in uso di un dispositivo, ad esempio una stampante o un'unità disco.  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Verificare che il dispositivo funzioni correttamente, quindi ripetere l'operazione.
