# <a name="constant-expression-not-representable-in-type-typename"></a>Espressione costante non rappresentabile nel tipo '\<typename>'
Si sta tentando di valutare una costante che non rientrerà nel tipo di destinazione, in genere perché sta causando un overflow dell'intervallo.  
  
 **ID errore:** BC30439  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1. Modificare il tipo di destinazione in modo che sia in grado di gestire la costante.
