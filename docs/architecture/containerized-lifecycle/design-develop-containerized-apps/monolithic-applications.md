## <a name="monolithic-application-deployed-as-a-container"></a><span data-ttu-id="733ec-146">Applicazione monolitica distribuita come contenitore</span><span class="sxs-lookup"><span data-stu-id="733ec-146">Monolithic application deployed as a container</span></span>
